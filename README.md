# 🏠 App Imobiliária Android

O **App Imobiliária Android** é um aplicativo desenvolvido para a plataforma Android utilizando **Java**, com o objetivo de simular a busca de imóveis de forma simples, intuitiva e organizada. O projeto foi criado com foco em estudos de desenvolvimento mobile, interface gráfica e Programação Orientada a Objetos (POO).
------------------------------------------------------------------

# 🚀 Funcionalidades:
- ✅ Tela inicial personalizada
- ✅ Seleção de cidade e estado
- ✅ Escolha da quantidade de quartos
- ✅ Escolha da quantidade de banheiros
- ✅ Seleção de garagem
- ✅ Escolha de comodidades do imóvel
- ✅ Tela resumo com todas as informações selecionadas
- ✅ Cálculo automático do valor estimado do aluguel
- ✅ Validação de campos obrigatórios
- ✅ Navegação entre Activities
------------------------------------------------------------------

# 🛠️ Tecnologias Utilizadas:
- **Java**
- **Android Studio**
- **XML**
- **POO (Programação Orientada a Objetos)**
- **Android SDK**
- **Gradle**
- **AndroidX**
------------------------------------------------------------------

# 📦 Como executar o projeto:

## Pré-requisitos:
- Android Studio instalado
- SDK Android configurado

## Passos:
1. Clone o repositório:
```bash
git clone https://github.com/andreysilva018/AppImobiliariaAndroid.git

 - Abra o projeto no Android Studio
 - Aguarde o Gradle sincronizar
 - Execute o aplicativo em um emulador ou dispositivo físico Android

💻 Como utilizar:
 - Abra o aplicativo
 - Informe a cidade e o estado desejado
 - Escolha as preferências do imóvel
 - Selecione quartos, banheiros e garagem
 - Visualize o resumo completo com o valor estimado do aluguel
```
